Tema 1. Variables

1. Reglas para Nombrar una Variable en PHP


En PHP, las variables son contenedores que almacenan datos. Para nombrar una variable de manera adecuada, debemos seguir algunas reglas:

Regla 1


El nombre de una variable debe comenzar con un signo de dólar ($), seguido por una letra o un guión bajo (_). Por ejemplo, $nombre o $edad son nombres de variables válidos.

Regla 2


Los nombres de variables son sensibles a mayúsculas y minúsculas, lo que significa que $nombre y $Nombre se consideran variables diferentes.

Regla 3


Puedes usar letras, números y guiones bajos en el nombre de una variable, pero no se permiten espacios ni caracteres especiales.

Regla 4


No se pueden utilizar palabras reservadas de PHP como nombres de variables.
Por ejemplo, no puedes nombrar una variable if o echo.



2. Snake Case, Upper Camel Case y Lower Camel Case


En PHP, es común seguir ciertas convenciones de nomenclatura para nombrar variables. Las tres más utilizadas son:

Snake Case


Las palabras del identificador se separan con guiones bajos y todas las letras se escriben en minúsculas.

Por ejemplo: $mi_variable_importante

Upper Camel Case


Las palabras se escriben juntas sin espacios y cada palabra comienza con una letra mayúscula.

Por ejemplo: $MiVariableImportante

Lower Camel Case


Similar al Upper Camel Case, pero la primera letra de la variable comienza en minúscula.

Por ejemplo: $miVariableImportante.

4. Concatenación de Variables


La concatenación de variables es la técnica de combinar múltiples valores en una cadena. En PHP, puedes concatenar variables utilizando el operador de concatenación (.)


5. Interpolación de Variables


PHP también permite la interpolación de variables dentro de cadenas de texto delimitadas por comillas dobles.
Esto significa que puedes insertar el valor de una variable directamente en una cadena.
